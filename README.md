# Rock Paper Scissors Game

A fun and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript. Players compete against the computer, with scores tracked in real time and instant win/lose/draw results displayed after each round.

## Features
- Interactive gameplay
- Random computer choices
- Real-time score tracking
- Responsive and clean UI
- Built with vanilla JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
